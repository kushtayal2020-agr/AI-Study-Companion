# 📚 AI Study Companion

## 📌 Description

AI Study Companion is a web-based academic productivity platform developed as a Project-Based Learning (PBL) project. It helps students manage their studies efficiently by combining planning, note-taking, progress tracking, and AI-powered assistance in one platform.

---

## 🎯 Objective

The objective of this project is to simplify student life by providing a centralized system where users can organize their study schedule, generate notes, track progress, and get instant help using AI.

---

## 🚀 Features

* 🧠 AI Chatbot for study assistance
* ✍️ AI-powered Notes Generator
* 📝 Notes management system
* 📅 Study planner for scheduling tasks
* 📊 Progress tracking dashboard
* 🔐 User authentication (Login & Signup)
* 🎨 Responsive and user-friendly interface

---

## 🤖 AI Integration

This project integrates AI using API keys to provide:

* Intelligent chatbot responses
* Automatic notes generation from user input

The AI functionality enhances learning by providing quick explanations and summarizing content efficiently.

---

## 🛠️ Technologies Used

* HTML5

## ⚙️ How to Run the Project

1. Open the project folder

2. Install dependencies:

```id="8slc3m"
npm install
```

3. Add your own OpenRouter API key before running locally.

	In server.js, set:

	const OPENROUTER_API_KEY = "YOUR_OPENROUTER_API_KEY";

	This key is required for both the chatbot and the notes generator features.

4. Start the server:

```id="6dks9c"
node server.js
```

5. Open your browser and go to:

```id="h23k9v"
http://localhost:3000
```
